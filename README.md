# Python Projects

This repository contains machine learning and GUI projects developed using Python.

## Projects

### Student Performance Predictor
Predicts student grades based on study hours and attendance.

### House Price Predictor
Predicts house prices using bedrooms, bathrooms, and house size.

### Employee Salary Predictor
Predicts employee salary using experience, education level, and working hours.

## Technologies Used

- Python
- Pandas
- Scikit-learn
- Tkinter
